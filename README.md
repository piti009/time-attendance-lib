📷 ระบบบันทึกเวลาเข้างานด้วยใบหน้าและ GPS (Face Recognition Attendance with GPS) เว็บแอปพลิเคชันสำหรับบันทึกเวลาเข้า-ออกงาน โดยใช้เทคโนโลยี จดจำใบหน้า (Face Recognition) ผ่านไลบรารี face-api.js ร่วมกับระบบ ตรวจสอบพิกัด (GPS Geofencing) ทำงานแบบ Serverless บน Google Apps Script และเก็บข้อมูลทั้งหมดลงใน Google Sheets โดยตรง

✨ ฟีเจอร์หลัก (Key Features) Serverless 100%: ไม่ต้องเช่าโฮสติ้ง รันบน Google Cloud (Apps Script) ฟรี

AI จดจำใบหน้า: ใช้ face-api.js ตรวจจับและเปรียบเทียบใบหน้าผ่านเบราว์เซอร์ แม่นยำด้วยค่าความเหมือน (Confidence Score)

จำกัดพื้นที่ลงเวลา (GPS Geofencing): ระบบจะอนุญาตให้ลงเวลาได้เฉพาะเมื่ออยู่ในรัศมีที่กำหนด (เช่น 100 เมตรจากออฟฟิศ)

Database Real-time: สร้างและบันทึกข้อมูลลง Google Sheets อัตโนมัติ (แยกแผ่นงาน Users, Attendance, Config)

ใช้งานง่าย:

ลงทะเบียน: ถ่ายรูปเก็บใบหน้าพนักงานได้เองผ่านหน้าเว็บ

สแกนเข้างาน: รวดเร็ว มีระบบยืนยันตัวตนพร้อมแสดงรูปถ่ายและคะแนนความเหมือน

ตั้งค่า: มีหน้า Admin สำหรับตั้งพิกัด GPS ของสถานที่ทำงาน

รองรับมือถือ: ออกแบบหน้าตา (UI) ให้สวยงามและใช้งานบนมือถือได้สะดวก (ใช้ฟอนต์ Sarabun)

🛠️ เทคโนโลยีที่ใช้ (Tech Stack) Frontend: HTML5, CSS3, JavaScript

AI Library: face-api.js (โหลดผ่าน CDN)

Backend: Google Apps Script (.gs)

Database: Google Sheets
